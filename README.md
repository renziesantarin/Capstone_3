# Capstone 3: Analysis with Power BI
Analysis by: Marenza Santarin

## Project Overview

This project analyzes EmporiUm’s East region sales data from 2022 to 2025 using Power BI. EmporiUm is a virtual student bookstore that includes both in-store and online sales. The goal of this report is to identify sales trends, category performance, state-level performance, and top-selling books.

## Business Goal

The purpose of this report is to help the East region sales team better understand sales performance and make more informed business decisions. The report is designed for regional directors, territory managers, and store managers who need a clear view of sales trends, strong-performing categories, state-level sales, and top-selling books.

## Tools Used

* Power BI Desktop
* Power Query
* DAX
* GitHub

## Data Sources

* Capstone3_Sample_Sales.xlsx
* book list.txt

## Key Questions

This report helps answer the following questions:

* How have sales changed over time?
* Which product categories are driving the most sales?
* Which states in the East region have the strongest sales performance?
* Which books are selling the most?
